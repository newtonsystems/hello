Requirements
============

Basic Requirements
------------------

- Python 2.7, 3.2, 3.3, 3.4, or 3.5

- Django (1.8.6+)

- djangorestframework

- python-dateutil

- markdown2

- beautifulsoup4

- Pygments


Django Test Requirements
------------------------
- tox

- django_dynamic_fixture

- django_nose

- pytest

- pytest-django

- pytest-xdist

- pytest-coverage

- nose-progressive

- responses>=0.4.0

- freezegun

Django Docs Requirements
-----------------------
- sphinx

- recommonmark

- sphinx-git

- sphinx_rtd_theme
